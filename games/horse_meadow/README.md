# Meadow Run

A peaceful 3D horse exploration prototype inspired by open-air adventure games, with no monsters and no fighting. Ride through hills, trees, flowers, water, clouds, and grazing horses.

## Run

```sh
python3 -m http.server 4173
```

Then open:

```text
http://localhost:4173
```

## Controls

- `WASD` or arrow keys: ride
- `Shift`: gallop while moving forward
- Mouse drag: rotate the camera
- `Space`: hop
- Double-click another horse: switch to riding it
- Xbox 360 controller: left stick rides, right stick rotates camera, `A` hops, right trigger gallops
